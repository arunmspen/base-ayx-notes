# base-ayx-notes

本项目用于归档和发布多个独立的 HTML 页面。这些页面本身不针对任何特定域名或网站，仅为通用型内容存档与展示。

## 项目简介

`base-ayx-notes` 是一个静态 HTML 页面集合仓库，主要用于存放各类笔记、演示页、工具页面等独立网页文件。每个 HTML 文件均为独立内容，不依赖后端服务，可直接在浏览器中打开查看。

## 目录结构

```
.
├── pages/            # 存放所有 HTML 页面文件
├── assets/           # 公共资源文件（样式、脚本、图片等）
├── index.html        # 项目首页/导航页（如有）
└── README.md         # 本文件
```

- `pages/` 目录下每个 HTML 文件对应一个独立页面，文件名建议简洁且有辨识度。
- `assets/` 用于存放多个页面共享的 CSS、JavaScript 或图片资源，减少重复。

## 页面归档说明

- 所有 HTML 文件均为独立、自包含的网页，可在本地直接双击运行。
- 页面之间无强制关联，可单独引用或分享。
- 归档内容可能随时间更新或移除，不保证所有页面长期可用。

## 维护说明

- 新增页面请放入 `pages/` 目录，并在 `index.html`（如有）中添加链接。
- 如有公共资源更新，请同步修改相关页面引用路径。
- 不建议在页面中嵌入外部资源（如第三方 CDN），以保持离线可用性。
- 提交前请确保 HTML 文件无语法错误，且链接、资源路径正确。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/base-ayx-notes.git
   ```
2. 进入项目目录，直接双击 `pages/` 下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务发布（需自行配置）。
